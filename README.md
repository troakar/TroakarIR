> **Моя личная завайбкоденная песочница для саунд-дизайна и физического моделирования, собранная из вдохновения, костылей и Taichi-тензоров.**

Это место, где академическая физика (FDTD, волновые уравнения, модуль Юнга) встречается с программированием в состоянии потока в 3 часа ночи. Проект будет пополняться генераторами, синтезаторами и инструментами **различной степени паршивости**: от пугающе реалистичных барабанов до скриптов, которые просто пытаются порвать вам диффузоры на мониторах (используйте лимитер!).

---

## 🎸 Что внутри (на данный момент)

Несмотря на хаотичную природу проекта, под капотом бьется довольно мощное сердце. 

*   **⚡ Taichi FDTD Engine (`core_taichi.py`)** — реалтайм расчет распространения звуковых волн в 2D-материалах на GPU (CUDA/Vulkan).
*   **🖐 Tactile Engine V6 (`tactile.py`)** — генератор микрофизики. Моделирует треск деревянных волокон, вязкое трение, хруст микротрещин и осыпание соли. 
*   **🏗 Система DLC (`dlc_loader.py`)** — да, я прикрутил оверинжиниринговую систему плагинов. Инструменты подгружаются динамически.
*   **📦 Multisample Packer** — скрипт, который собирает весь сгенерированный хаос в готовые `.multisample` патчи для Bitwig (и других сэмплеров) с разбивкой по Velocity и Round Robin.

### 🥁 Актуальные DLC (Инструменты)

#### 1. The Dhol (Кавказский Дхол)
Полностью синтезированный с нуля кавказский барабан. 
*   Две мембраны (Бас и Тэк), связанные через акустический объем воздуха.
*   Гетерогенные сетки материалов (можно подмешать в деревянную кадушку вкрапления металла или соли).
*   Встроенные колокольчики (Shkhshkhan), которые звенят в зависимости от физического ускорения кадушки.
*   Интеграция с PyRoomAcoustics для позиционирования в 3D-комнате.

## 🛠 Установка и запуск

Проект написан на Python. Для генерации звука в приемлемые сроки **очень желательно наличие видеокарты** (Taichi сам подхватит CUDA или Vulkan, в крайнем случае упадет на CPU).

1. **Клонируем песочницу:**
   ```bash
   git clone https://github.com/ТВОЙ_НИК/troakar-lab.git
   cd troakar-lab
   ```

2. **Ставим зависимости:**
   ```bash
   pip install numpy scipy pyroomacoustics taichi Pillow tkinterdnd2
   ```

3. **Запускаем лабораторию:**
   ```bash
   python main.py
   ```
