# Taichi FDTD Impulse Response Generator

## Описание
Модуль `@ui/tab_taichi.py` предоставляет графический интерфейс для генерации импульсных характеристик (IR) музыкальных инструментов и объектов с использованием метода FDTD (Finite-Difference Time-Domain) на базе движка Taichi. Модуль моделирует физические процессы распространения звуковых волн в двумерных материалах с заданными геометрическими и акустическими свойствами.

Особенностью реализации является визуальное отображение инструментов и объектов в **разрезе**, что позволяет детально анализировать распространение волн внутри структуры. Это дает уникальную возможность изучать акустические свойства не только поверхности, но и внутренней структуры материалов и форм.

## Принципы работы алгоритмов FDTD

### Основы метода FDTD
Метод конечных разностей во временной области (FDTD) — это численный метод решения дифференциальных уравнений в частных производных, широко используемый в акустике и электродинамике. В данном проекте он применяется для моделирования распространения звуковых волн в двумерной среде.

**Ключевые принципы:**
1. **Дискретизация пространства и времени**: Расчетная область разбивается на сетку 128x128 точек, а время дискретизируется с шагом, соответствующим частоте дискретизации 44.1 кГц.

2. **Волновое уравнение**: Решается двумерное волновое уравнение:
   ```
   ∂²p/∂t² = c² (∂²p/∂x² + ∂²p/∂y²) - σ ∂p/∂t
   ```
   где:
   - `p` — акустическое давление,
   - `c` — скорость звука в материале,
   - `σ` — коэффициент затухания.

3. **Граничные условия**: На границах инструмента применяются условия свободного поля, а на границах расчетной области — поглощающие граничные условия для предотвращения отражений.

4. **Возбуждение**: Моделируется двумя способами:
   - **Импульсное возбуждение** (удар): Кратковременный импульс в точке возбуждения.
   - **Фрикционное возбуждение** (смычок): Шумовой сигнал, имитирующий трение.

### Связь звука и материала
Физические свойства материала напрямую влияют на характер генерируемого звука:

1. **Скорость звука (`c`)**:
   - Определяет высоту основного тона инструмента.
   - Чем выше скорость звука, тем выше частота резонансов.
   - Формула: `c = sqrt(E / ρ)`, где `E` — модуль Юнга, `ρ` — плотность.

2. **Плотность (`ρ`)**:
   - Влияет на инерционность системы и амплитуду колебаний.
   - Более плотные материалы генерируют более глухой и тихий звук.

3. **Коэффициент затухания (`σ`)**:
   - Определяет продолжительность затухания звука.
   - Высокий коэффициент приводит к быстрому затуханию высоких частот.

4. **Толщина материала**:
   - Влияет на жесткость и резонансные свойства.
   - Тонкие материалы резонируют на более высоких частотах.

5. **Геометрия инструмента**:
   - Форма определяет распределение мод колебаний.
   - Размер влияет на основную частоту и спектральный состав.

### Процесс генерации IR
1. **Инициализация расчетной сетки**:
   - Создается двумерная сетка 128x128 точек.
   - Задаются физические свойства материала в каждой точке.

2. **Задание геометрии**:
   - На основе выбранного инструмента генерируется маска, определяющая форму инструмента.
   - Точки вне маски считаются свободным пространством.

3. **Настройка возбуждения**:
   - Для ударного режима: импульсное возбуждение в заданной точке.
   - Для фрикционного режима: шумовой сигнал в точке смычка.

4. **Моделирование распространения волн**:
   - На каждом шаге по времени обновляются значения акустического давления в каждой точке сетки.
   - Учитываются граничные условия и затухание.

5. **Съем сигнала**:
   - В заданной точке съема записывается временная зависимость акустического давления.
   - Этот сигнал и является импульсной характеристикой.

6. **Сохранение результата**:
   - Сигнал сохраняется в формате WAV (32-бит float, 44.1 кГц).

### Особенности реализации
1. **Визуализация тепловизора**:
   - Во время расчета отображается тепловизор Taichi, показывающий распределение акустического давления в реальном времени.
   - Это позволяет визуально контролировать процесс распространения волн.

2. **Масштабирование**:
   - Геометрия инструмента может масштабироваться от 0.3x до 3.0x.
   - Масштабирование влияет на физические размеры и толщину материала, что изменяет резонансные свойства.

3. **Точки возбуждения и съема**:
   - Пользователь может задавать произвольные точки возбуждения и съема.
   - Положение точек влияет на спектральный состав и тембр звука.

4. **Длительность рендера**:
   - Позволяет контролировать продолжительность импульсной характеристики.
   - Длинные IR захватывают больше резонансов и затуханий.

## Примеры влияния параметров на звук

| Параметр               | Влияние на звук                                                                 |
|------------------------|---------------------------------------------------------------------------------|
| Скорость звука ↑       | Более высокий основной тон, ярче высокие частоты                                |
| Плотность ↑            | Глухой, тихий звук с акцентом на низкие частоты                                 |
| Затухание ↑            | Быстрое затухание, короткий реверберационный хвост                              |
| Толщина ↑              | Более жесткий звук с акцентом на низкие частоты                                 |
| Масштаб ↑              | Более низкий основной тон, увеличенная продолжительность затухания             |
| Точка возбуждения      | Изменение тембра: ближе к краю — ярче высокие частоты, ближе к центру — глуше   |
| Точка съема            | Изменение баланса частот: ближе к возбуждению — ярче атака, дальше — мягче      |

## Ограничения модели
1. **Двумерность**: Модель не учитывает трехмерные эффекты, такие как распространение волн в толще материала.
2. **Линейность**: Не учитываются нелинейные эффекты, характерные для реальных материалов при больших амплитудах.
3. **Граничные условия**: Упрощенные граничные условия могут не полностью соответствовать реальным физическим процессам.
4. **Разрешение сетки**: Ограниченное разрешение (128x128) не позволяет моделировать высокочастотные детали.

## Рекомендации по использованию
1. Для ударных звуков используйте импульсное возбуждение с короткой длительностью IR (0.5–1.5 сек).
2. Для фрикционных звуков (смычок) используйте шумовое возбуждение с длительностью IR 2–4 сек.
3. Экспериментируйте с положением точек возбуждения и съема для получения разнообразных тембров.
4. Масштабируйте геометрию для изменения основного тона инструмента.
5. Используйте материалы с высокой скоростью звука для ярких, высокочастотных звуков и материалы с высокой плотностью для глухих, низкочастотных звуков.
